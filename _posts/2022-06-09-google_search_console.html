구글에 내 블로그를 검색했을 때 트래픽 및 실적을 측정하고 검색 결과 노출에 도움을 주는 Google Search Console을 적용해보자.

※ 필자는 jekyll-sitemap이 설정되어 있는 상태입니다.

1.

시작하기 Click!

2.

URL 접두어를 선택하였고, URL 입력 후 계속

도메인

모든 하위 도메인(m, www 등)과 여러 프로토콜(http, https, ftp)을 포함하며 DNS 레코드 인증만 지원한다.
ex.
  • http://example.com/dresses/1234 (o)
  • https://example.com/dresses/1234 (o)
  • http://www.example.com/dresses/1234 (o)
  • http://support.m.example.com/dresses/1234 (o)

URL 접두어

프로토콜(http/https)을 포함하여 지정된 접두사가 있는 URL만 포함하며 다양한 인증 방식을 지원하지만 필자는 HTML 태그 인증으로 진행했다.
ex.
  • http://example.com/dresses/1234 (o)
  • https://example.com/dresses/1234 (x) → http가 일치하지 않음
  • http://www.example.com/dresses/1234 (x) → www.이 일치하지 않음

3.

다운로드한 파일을 최상위 디렉토리에 올린 뒤에 확인

4.

5.

Sitemap 입력 후 제출

7.

8.
