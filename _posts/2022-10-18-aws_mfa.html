AWS MFA란?

MFA(Multi-Factor Authentication), 말 그대로 다중인증이다. AWS 로그인 시 2가지 이상의 인증을 해야한다.
AWS에 대한 보안을 강화하지 않으면 해킹을 당할 시 비트코인 채굴 등으로 엄청난 과금이 될 수 있기 때문에 보안 강화는 필수적이다.

해당 포스팅은 AWS ROOT 계정으로 설정하는 내용입니다.

1. 앱스토어에서 google authenticator 검색 후 설치

1. AWS 로그인을 하고 우측 상단에 이름 클릭 후 보안 자격 증명 클릭

2. MFA 할당 클릭

3. 인증 방식 선택(모바일 디바이스 방식)

4. QR 코드 표시 클릭

5. 1번에서 설치했던 google authenticator에 QR 코드 인식 후 생성된 인증코드 입력

6. 설정 확인을 위해 ROOT 계정으로 로그인

7. MFA 인증코드 입력
